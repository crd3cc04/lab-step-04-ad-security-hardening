# Active Directory Security Hardening & Logging Lab

## Overview
This lab builds on an existing Active Directory environment by implementing enterprise-grade security hardening, advanced auditing, SIEM integration, attack simulations, and SOC-style detection and response. The goal is to demonstrate practical Security+ skills and real-world defensive security workflows.

## Objectives
- Apply security baseline configurations through Group Policy
- Enable advanced auditing for authentication, authorization, and system activity
- Forward logs to a SIEM for monitoring and analysis
- Simulate common attack techniques and capture event data
- Build detection rules and dashboards
- Document findings in a security baseline report
