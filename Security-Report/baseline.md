# Security Baseline Report

## Overview
This report summarizes the security posture of the Active Directory environment before and after hardening.

## Findings
- Identified vulnerabilities
- Misconfigurations
- Missing audit controls
- Weak authentication policies

## Hardening Actions Taken
- Password and lockout policies enforced
- Advanced auditing enabled
- Firewall rules applied
- Security options hardened

## Evidence
- Screenshots
- Event logs
- SIEM alerts

## Recommendations
- Additional hardening steps
- Monitoring improvements
- Long-term security considerations

## Summary
The environment now meets baseline security expectations for enterprise Windows domains and aligns with Security+ best practices.
