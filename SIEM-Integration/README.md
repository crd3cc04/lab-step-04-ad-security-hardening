# SIEM Integration

## Purpose
This section documents the integration of the Active Directory environment with a SIEM platform to enable centralized log collection, monitoring, and alerting.

## Components
### 1. Agent Setup
Located in `Agent-Setup/`:
- Installation steps for the SIEM agent
- Configuration details
- Verification steps

### 2. Log Forwarding
Located in `Log-Forwarding/`:
- Security logs
- System logs
- PowerShell logs
- Kerberos events
- Process creation events

### 3. Screenshots
Located in `Screenshots/`:
- Agent status
- Logs arriving in SIEM
- Example queries

## Outcomes
- Centralized visibility into authentication events
- Ability to detect brute-force attempts
- Monitoring of privilege escalation and suspicious activity
