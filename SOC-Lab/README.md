# SOC Lab

## Purpose
This section demonstrates SOC-style monitoring, detection, and incident response using logs forwarded from the Active Directory environment.

## Components
### 1. Dashboards
Located in `Dashboards/`:
- Failed logons
- Process creation
- Group changes
- PowerShell activity

### 2. Detection Rules
Located in `Detection-Rules/`:
- Brute-force detection
- Suspicious PowerShell
- Privilege escalation
- Unauthorized access

### 3. Incident Response
Located in `Incident-Response/`:
- Attack timeline
- Evidence collected
- Root cause analysis
- Recommendations

## Outcomes
- Ability to detect common attack patterns
- Hands-on experience with SIEM analysis
- Realistic incident response workflow
